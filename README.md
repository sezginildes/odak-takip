# 🎯 Odak Takip — Günlük Gözlem Formu

A lightweight daily observation form for tracking focus, attention, impulsivity, sleep, and study behavior. Built for educational counseling workflows.

**Live →** [sezginildes.github.io/odak-takip](https://sezginildes.github.io/odak-takip)

---

## What It Does

A structured daily form that captures behavioral observations across 6 areas:

- **Morning & Routine** — wake time, readiness, clothing organization
- **Study Behavior** — whether studied, focus duration, screen interference, desk organization
- **Impulsivity & Behavior** — interrupting, sudden reactions, response to reminders
- **General State** — mood, energy, motivation, attitude toward school
- **Open Notes** — concerns, positives, free observations

Responses are submitted directly to the counselor via [Formspree](https://formspree.io).

---

## Why It Was Built

Standard counseling tools rarely capture *daily* behavioral patterns at home. This form bridges the gap between school observations and home context — giving counselors a richer, evidence-based picture of the student across environments.

Particularly useful when:
- Tracking focus and impulsivity patterns over time
- Building a behavioral record for specialist referrals (e.g., ADHD assessment)
- Supporting parent-counselor collaboration

---

## Stack

- Pure HTML / CSS / JS — no dependencies
- [Formspree](https://formspree.io) for form submission
- Hosted on GitHub Pages

---

## Usage

1. Open the form daily
2. Fill in the student's name and today's date
3. Answer the 19 questions (takes ~2 minutes)
4. Submit — responses go directly to the counselor
